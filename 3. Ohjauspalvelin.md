# 3. Ohajauspalvelin

## Tehtävänanto

1. Luodaan ohjauspalvelin toimialueelle
2. Asennetaan Windows-käyttöjärjestelmä luotuun virtuaalikoneeseen
3. Luodun virtuaalikoneen mallinnus
4. Mallikoneen käyttäminen uusien virtuaalikoneiden pohjana

##  Ohjauspalvelimen rungon määritys

Aloitin kirjautumalla Azure Labsiin ja avaamalla virtuaalikoneen. Seuraavaksi ryhdyin luomaan uutta virtuaalikonetta hyödyntäen aikaisemmassa harjoituksessa luotua virtuaalikovalevyä "WindowsServer_DC.vhdx". Avasin Hyper-V managerin ja valitsin "New -> Virtual Machine
