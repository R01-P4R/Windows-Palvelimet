# 1. Virtuaalikoneiden käyttöalusta Hyper-V

## Tehtävänanto

1. Asennetaan Hyper-V Hypervisor Azure Labs ympäristöön. Hyper-V ohjelman avulla asennetaan ja hallitaan virtuaalisia koneita ja verkkoa.

2. Määritetään ympäristön verkkoasetukset. Tarvisemme virtuaalisen verkon, jotta laitteemme voivat olla yhteydessä toisiinsa ja internetiin.

3. Laaditaan dokumentti ympäristön työpöydälle, johon tallennetaan omat konfiguraatiot (ip-osoitteet, tunnuset, salasanat, jne.)


### Laitteisto
 
* Käyttöjärjestelmä	Microsoft Windows 10 Enterprise LTSC 64 bit
* Prosessori i5-6600
* RAM 16 GB
* Virtuaaliympäristö : Azure Labs Service 
* Virtuaalikoneen käyttöjärjestelmä: Microsoft Windows Server 2019 Datacenter (Version	10.0.17763 Build 17763)
* Virtuaalikoneen RAM: 16 GB


Aloitin asentamalla ohjeen mukaan Hyper V-virtuaalikytkimen painamalla virtuaalikoneen työpöydällä olevaa asennuskuvaketta, virtuaalikoneeseen täytyi ottaa yhteys uudestaan asennuksen päätyttyä. Yhteyden palattua avautui Hyper-V manager-ohjelma.
