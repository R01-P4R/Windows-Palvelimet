# 5. Toimialueen resurssit

## Tehtävänanto

1. Tutustutaan aktiivihakemistoon
2. Käyttäjien lisäys aktiivihakemistoon
3. Määritetään ohjauspalvelimen staattinen IP-osoite
4. Määritetään ohjauspalvelimen nimi ja poistetaan lukitusruutu-ominaisuus pois käytöstä
5. Aktivoidaan ohjauspalvelimen käyttöjärjestelmä
6. Asennetaan ohjauspalvelimelle tarvittavat roolit ja ohjelmisto

<br>

## Aktiivihakemiston sisältöön tutustuminen


Aloitin kirjautumalla Azure ympäristöni ohjauspalvelimelleni WindowsServer_DC ja Server Manager sovelluksesta valitsin Tools-valikosta: "Active Directory Administrative Center". Tästä avautui Active Directoryn hallintapaneeli, jolla voi määrittää koko toimialueen säännöt keskitetysti yhdeltä koneelta (Ohjauspalvelimelta WindowsServer_DC tässä tapauksessa) . Aktiiviihakemisto on hakemistopalvelu, mikä muodostaa tietokannan. Aktiivihakemisto sisältää eri säiliöitä (Container) esim. Computers, System Users, ja Program Data.

<br>
